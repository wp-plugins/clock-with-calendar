== Description ==

Bfcshop Wordpress Clock Calendar Widget gives you the ability to add a flash based clock and calendar to your Wordpress blog. 


== Installation ==

1. Download the zip file and extract the contents.
2. Make sure you're running WordPress version 2.3 or better. It won't work with older versions.
3. Upload the 'bfcshop-wordpress-clock-calendar' folder to your plugins directory (plugins/bfcshop-wordpress-clock-calendar).
4. Activate the plugin through the 'plugins' page in WP.
5. See widget options to adjust things like display size...


== Frequently Asked Questions ==
The plugin is not working correctly and is not showing up?

Please download the ZIP file from this URL: http://www.spletne-kode.si/clock-with-calendar.rar Extract and upload the directory to your plugin directory,
activate it trough 'plugins' and set the size and title in Widget section.
